# dlna-demo

## 第一阶段，查看参考资料，实现向电视端投屏一张图片。
  投屏一张图片，实现功能比较简单，基本上就是扫描电视，向电视端发起执行action，还不涉及到事件部分，可以作为学习upnp协议的入门demo。
